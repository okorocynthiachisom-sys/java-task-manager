# Java Task Manager

This is a simple Java console application that allows users to manage tasks.

## Features
- Add tasks
- View tasks
- Exit program

## How it works
The program runs in a loop and displays a menu.  
Users choose an option, and the program performs an action.

## Technologies Used
- Java
- ArrayList
- Scanner
- Loops

## What I Learned
- Handling user input using Scanner
- Storing data dynamically using ArrayList
- Using loops to control program flow
- Organizing code using methods

## How to Run
1. Compile:
   javac Main.java

2. Run:
   java Main
